# Tip Calculator

A clean, responsive **Tip Calculator web application** built using **HTML, CSS, and Vanilla JavaScript**.  
The app allows users to calculate tip and total amount per person with real-time updates and a modern UI.

---

## Features

- Enter bill amount with validation
- Select tip percentage or enter custom tip
- Split bill among multiple people
- Real-time calculation (no calculate button)
- Reset functionality
- Dark mode with persistent theme
- Fully responsive design
